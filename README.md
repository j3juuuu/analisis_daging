# Sistem Analisis Kesegaran Daging

Aplikasi Flask sederhana untuk menganalisis kesegaran daging menggunakan fitur warna (RGB & HSV), tekstur (GLCM), dan bentuk (area & perimeter).

## Persyaratan
- Python 3.8+
- paket: lihat `requirements.txt`

## Instalasi

Jalankan perintah berikut di PowerShell atau terminal:

```powershell
python -m pip install --upgrade pip
pip install -r requirements.txt
```

## Menjalankan

```powershell
cd C:\Users\LENOVO\Downloads\analisis_daging
python app.py
```

Buka web di: http://127.0.0.1:5000

## Keterangan
- Maksimum ukuran upload: 5 MB
- Lokasi upload: `static/uploads`
